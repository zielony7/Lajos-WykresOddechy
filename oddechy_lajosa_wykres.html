⚪ ⚙ Połączenie z Google Sheets

Jednorazowa konfiguracja:
1. Otwórz Oddechy Po Edycji w Sheets na komputerze
2. Menu → Rozszerzenia → Apps Script → usuń stary kod → wklej KOD_DO_APPS_SCRIPT.txt
3. Wdróż → Nowe wdrożenie → Aplikacja internetowa → Uruchom jako: Ja · Dostęp: Wszyscy
4. Autoryzuj → skopiuj URL → wklej poniżej → Zapisz i połącz
5. Kliknij Wyślij bazę do Sheets (tylko raz!)

Jak dodawać nowe oddechy:
Otwórz Surowe Oddechy Z Aplikacji (Docs) → wklej surowy tekst z aplikacji → zapisz.
Następnym razem gdy otworzysz ten wykres — dane zostaną automatycznie pobrane i Docs wyczyszczony. ✓
